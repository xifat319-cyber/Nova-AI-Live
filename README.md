# Nova AI Live 🚀

An advanced, real-time AI Scholar Assistant built with Python and Google Gemini.

## 🌟 Overview
Nova AI is designed to act as a smart scholar assistant, providing real-time, intelligent responses using the power of Google's Gemini AI model. This repository contains the backend code and frontend templates for the live web application.

## 🛠️ Tech Stack
* **Backend:** Python
* **AI Model:** Google Gemini API
* **Frontend:** HTML/CSS (Jinja Templates)
