# Nova AI Live

> A real-time AI Scholar Assistant powered by Google Gemini — built to answer, explain, and assist with anything academic.

---

## What is Nova AI?

Nova AI is a web-based intelligent assistant designed for students and researchers. It uses Google's Gemini model under the hood to deliver fast, context-aware responses through a clean, browser-based interface. Whether you need help understanding a concept, summarizing content, or getting instant answers — Nova has you covered.

## Getting Started

### Prerequisites

- Python 3.8+
- A valid [Google Gemini API key](https://makersuite.google.com/app/apikey)

### Installation

```bash
# Clone the repository
git clone https://github.com/xifat319-cyber/Nova-AI-Live.git
cd Nova-AI-Live

# Install dependencies
pip install -r requirements.txt

# Add your Gemini API key
# Create a .env file and add:
GEMINI_API_KEY=your_api_key_here

# Run the application
python app.py
```

Then open your browser and go to `http://localhost:5000`
